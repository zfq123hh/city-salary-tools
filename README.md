# 秒算工资工具 MVP

中文程序化 SEO 工具站：城市税后工资、社保、公积金、个税、贷款和日期工具。

## 当前 MVP

- 16 个核心城市
- 每个城市 3 个长尾页面：税后工资、社保、公积金
- 9 个通用工具页面：年终奖个税、退休年龄、房贷、车贷、存款、复利、工作日、年龄、BMI
- 首页、城市索引、工具索引、隐私政策、条款、联系、推广披露
- 自动生成 `sitemap.xml`、`robots.txt`、结构化数据、OpenGraph
- 无后端、无数据库、无依赖，适合 GitHub Pages / Cloudflare Pages / 静态服务器
